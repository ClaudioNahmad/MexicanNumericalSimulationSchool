NAME

       clearrot - resets the viewing angle down the z-axis

SYNOPSIS

       clearrot

ALIASES

       crot

DESCRIPTION

       clearrot  resets  the viewing angle to be looking down the z-axis.  To
       view the particles from the new direction the clearrot command must be
       followed  by either the view command to maintain the same plot type or
       by viewall, viewarray, viewgas, viewmag, viewpot, viewrad, or viewstar
       commands.

SEE ALSO

       clearall(1),  resetcolor(1),  rotate(1),  unloadall(1),  unmarkall(1),
       unzoom(1)
