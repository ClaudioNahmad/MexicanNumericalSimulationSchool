NAME

       printhelp - print useful information about commands on the printer

SYNOPSIS

       printhelp command

ALIASES

       phelp

DESCRIPTION

       printhelp prints useful information about command on the printer.  For
       general information about TIPSY type ''printhelp tipsy  <return>''  or
       just  ''printhelp  <return>''.  To print the information on the screen
       use the help command.  To print a short description  of  a  particular
       command  use  the whatis command.  To print a brief description of all
       the commands use the commands command.  To find the correct  arguments
       for a command use the arguments command.

SEE ALSO

       arguments(1), commands(1), help(1), whatis(1)
