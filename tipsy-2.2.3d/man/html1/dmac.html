NAME

       deletemacro - delete a macro

SYNOPSIS

       deletemacro macro-name

ALIASES

       dmac

DESCRIPTION

       deletemacro deletes the macro macro-name from the macro list.  Macro's
       are created with the makemacro command or read in using the  readmacro
       command.  If no macro exists with the name macro-name an error is gen-
       erated.

SEE ALSO

       macro(1), makemacro(1), readmacro(1)
