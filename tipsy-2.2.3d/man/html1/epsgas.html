NAME

       gravsoftgas - set the gravitational softening for the gas and stars

SYNOPSIS

       gravsoftgas epsgas

ALIASES

       epsgas

DESCRIPTION

       gravsoftgas  sets  the value of the gravitational softening length for
       the gas and star particles to epsgas for use in commands such as rota-
       tioncurve or viewgas.  The softening length is in system units.
