NAME

       writebox - write particles in a box to a file

SYNOPSIS

       writebox box_number file_name [mark]

ALIASES

       wbox

DESCRIPTION

       writebox  writes all the particles in box box_number to file file_name
       in TIPSY ASCII format.  If the optional argument mark is  given,  only
       the marked particles will be written to the file.
