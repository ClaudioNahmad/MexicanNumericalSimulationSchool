NAME

       deletewin - delete a window

SYNOPSIS

       deletewin window-number

DESCRIPTION

       deletewin  deletes  a  window window-number that was created using the
       window command.  If window-number does not exist an  error  is  gener-
       ated.

SEE ALSO

       hard(1), window(1)
