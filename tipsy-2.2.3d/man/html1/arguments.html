NAME

       arguments - list the arguments required for a command

SYNOPSIS

       arguments command

DESCRIPTION

       Lists  the  arguments required to run command.  More detailed informa-
       tion can be found using the help command.

SEE ALSO

       commands(1), help(1), printhelp(1), whatis(1)
