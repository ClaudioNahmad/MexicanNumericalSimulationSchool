NAME

       axesscale - set the length of plotted coordinate axes

SYNOPSIS

       axesscale scale

ALIASES

       ascale

DESCRIPTION

       axesscale  sets  the  length  of the plotted coordinate axes to scale.
       The initial value of scale is 1.  Whether or not coordinate  axes  are
       plotted is controlled with the showaxes command.

SEE ALSO

       showaxes(1)
