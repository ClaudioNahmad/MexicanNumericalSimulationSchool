NAME

       closeascii - close the currently open ascii file

SYNOPSIS

       closeascii

DESCRIPTION

       closeascii  closes the currently open ascii file.  If no ascii file is
       open, an error is generated.  Ascii files  can  be  opened  using  the
       closeascii  command.   Binary  files  are  opened and closed using the
       openbinary and closebinary commands.

SEE ALSO

       closebinary(1), openascii(1), openbinary(1)
