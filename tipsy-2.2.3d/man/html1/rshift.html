NAME

       redshift - set the redshift and coordinate system

SYNOPSIS

       redshift z coordinates hubble-constant omega boundary period

DESCRIPTION

       redshift  sets the value of the redshift to z for use in commands such
       as vista.  coordinates tells TIPSY if the simulation used comoving  or
       physical  coordinates.  If coordinates is "physical" or "p" this means
       physical coordinates were used and if coordinates is "comoving" or "c"
       this  means  comoving  coodinates  were  used.  hubble-constant is the
       value of the hubble constant in  km/sec/Mpc.   boundary  concerns  the
       boundary  conditions.   If boundary is "vacuum" or "v" vacuum boundary
       conditions are used and period is ingored. If boundary  is  "periodic"
       or  "p"  then  periodic  boundary  conditions  are  used with a period
       period.
