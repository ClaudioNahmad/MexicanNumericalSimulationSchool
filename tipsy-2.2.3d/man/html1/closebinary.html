NAME

       closebinary - close the currently open binary file

SYNOPSIS

       closebinary

DESCRIPTION

       closebinary  closes the currently open binary file.  If no binary file
       is open, an error is generated.  Binary files are opened using  either
       the  openbinary  or  readascii  commands.   Ascii files are opened and
       closed using the openascii and closeascii commands.

SEE ALSO

       closeascii(1), openascii(1), openbinary(1), readascii(1)
