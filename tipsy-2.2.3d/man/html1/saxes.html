NAME

       showaxes - turn axes display off or on

SYNOPSIS

       showaxes yes/no

ALIASES

       saxes

DESCRIPTION

       showaxes  takes a yes or no argument.  The yes argument causes coordi-
       nate axes to be plotted in green in plots. The no argument causes  the
       axes  not  be  plotted.  This is the default.  The arguments yes or no
       can be abbreviated y or n.  The size of the axes are controlled  using
       the axesscale command.

SEE ALSO

       showvec(1), showvel(1)
