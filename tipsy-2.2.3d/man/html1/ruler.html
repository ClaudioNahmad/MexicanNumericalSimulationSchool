NAME

       ruler - find projected distance between two points

SYNOPSIS

       ruler

DESCRIPTION

       ruler is used to find the projected distance between two points in the
       active plot.  If the currently plotted box is not the currently active
       box,  or if the internal state has been altered since the last plot an
       error is generated.

       Once the command is executed a pointer will appear on  the  view  sur-
       face.    Click the left mouse button on one point, then click it again
       on a second point.  The projected distance will be printed.
