NAME

       markmass - mark particles according to their mass

SYNOPSIS

       markmass box-number minimum maximum

DESCRIPTION

       markmass  marks  all  the particles in box box-number that have masses
       between minimum and maximum, inclusive.   If  box  box-number  is  not
       loaded an error is generated.

       To  unmark  particles  use either the unmarkall or unmarkbox commands.
       Remember, marked particles are always  plotted  in  green.   Particles
       remain  marked  even  when  a new time frame is loaded but after a new
       time frame is loaded  any particles that were marked in  the  previous
       time  frame  can only be unmarked using the unmarkall command.  Marked
       particles can be written to a file using the writemark command or read
       from  a  file using the readmark command. Particles can also be marked
       using the markarray, markbox and markgal commands.   Statistics  about
       marked particles can be determined using the markstat command.
