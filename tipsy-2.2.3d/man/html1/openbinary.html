NAME

       openbinary - open a binary file

SYNOPSIS

       openbinary filename

ALIASES

       openb

DESCRIPTION

       openbinary  is  a  command that opens a binary file of the name, file-
       name.  Once a binary file is open a specific time must still be loaded
       using  the  loadbinary  command.   If a binary file is already open an
       error is generated.  Binary files can be closed using the  closebinary
       command.   Binary  files can be opened and created using the readascii
       command.  Binary files can also be created using the ascii2binary pro-
       gram.  Ascii  files  are  opened  and  closed  using the openascii and
       closeascii commands.

SEE ALSO

       closeascii(1),    closebinary(1),     loadbinary(1),     openascii(1),
       readascii(1)
