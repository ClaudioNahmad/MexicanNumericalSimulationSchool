NAME

       setsphere - make and load a spherical "box" from the command line

SYNOPSIS

       setsphere box xcenter ycenter zcenter radius

DESCRIPTION

       setsphere  is used to define the boundaries of box through the command
       line.  The requested box must be between 1 and MAXBOX or an error will
       be  generated.   The value of MAXBOX can be found by using the version
       command.  The box is a sphere of radius radius centered on the coordi-
       nates (xcenter, ycenter, zcenter).

       Once  a box is made and loaded it still isn't the active box for plot-
       ting until the activatebox command is used.  Boxes are  removed  using
       the unload command.

       Boxes  with  numbers  greater than box 0 cannot be made until box 0 is
       loaded.  Boxes can also be made and loaded using the makebox and  set-
       box  commands.   Box  0  can also be made and loaded using the loadall
       command.

SEE ALSO

       activatebox(1), loadall(1),  makebox(1),  setbox(1),  unload(1),  ver-
       sion(1)

BUGS

       The  volume of the box is not done correctly in commands such as boxs-
       tat and the box is not plotted correctly when showboxes is yes if  the
       box is spherical.
