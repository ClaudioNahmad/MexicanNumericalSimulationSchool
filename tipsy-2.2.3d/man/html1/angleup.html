NAME

       angleup - change the viewing angle to edge-on

SYNOPSIS

       angleup box-number particle

ALIASES

       aup

DESCRIPTION

       angleup  changes the viewing angle so that the angular momentum vector
       of particle type particle in box box-number  points  straight  up  and
       lies  in  the plane of the screen.  Remember that the angular momentum
       vector of any box excludes the center of mass motion of the  box.   To
       view  the particles from the new direction the angleup command must be
       followed by either the view command to maintain the same plot type  or
       by viewall, viewarray, viewgas, viewmag, viewpot, viewrad, or viewstar
       commands.  If the box box-number is not loaded an error is  generated.

OPTIONS

   Particle
       The possible choices for particle are:

              all
                 Use  the angular momentum vector of all the particles in box
                 box-number

              dark
                 Use the angular momentum vector of only the dark matter par-
                 ticles in box box-number

              gas
                 Use the angular momentum vector of only the gas particles in
                 box box-number

              star
                 Use the angular momentum vector of only the  star  particles
                 in box box-number
