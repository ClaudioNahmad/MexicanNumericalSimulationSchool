NAME

       help - display useful information about commands on the screen

SYNOPSIS

       help command

ALIASES

       man

DESCRIPTION

       help  displays  useful  information  about command on the screen.  For
       general information about TIPSY type ''help tipsy <return>''  or  just
       ''help <return>''.  To print a hard copy use the printhelp command. To
       print a short description of a particular command use the whatis  com-
       mand.   To  print a brief description of all the commands use the com-
       mands command.  To find the correct arguments for a  command  use  the
       arguments command.

SEE ALSO

       arguments(1), commands(1), printhelp(1), whatis(1)
