NAME

       smooth - set parameters for smoothing

SYNOPSIS

       smooth ball_size n_smooth

DESCRIPTION

       Smooth  sets  the value of the ball size to ball_size or the number of
       particles to smooth over to n_smooth for use in commands such as vista
       or  absorb  that  require  estimating  smooth  field  parameters.   If
       ball_size  is  non-negative,  n_smooth  is  ignored.   Otherwise,  the
       smoothing  length  for each particle is adjusted such that it contains
       n_smooth particles.
