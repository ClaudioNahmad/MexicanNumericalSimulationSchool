NAME

       commands - list the possible commands

SYNOPSIS

       commands [print]

DESCRIPTION

       commands  lists the possible commands with a short description of each
       command.  The command usually takes no arguments unless the command is
       given with the option print.  In this case the listing will be made on
       the printer.  More detailed information can be found  using  the  help
       command.

SEE ALSO

       arguments(1), help(1), printhelp(1), whatis(1)
