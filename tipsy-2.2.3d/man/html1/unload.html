NAME

       unload - unload a box

SYNOPSIS

       unload box

DESCRIPTION

       unload  is  used to unload a box.  The requested box must be between 0
       and MAXBOX.  If the box is marked, then it will be unmarked.

       activatebox(1), loadall(1), makebox(1), setbox(1), setsphere(1), show-
       boxes(1)
