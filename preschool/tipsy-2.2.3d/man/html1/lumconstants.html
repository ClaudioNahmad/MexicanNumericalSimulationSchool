NAME

       lumconstants - set the constants related to stellar luminosities

SYNOPSIS

       lumconstants msolunit kpcunit

ALIASES

       lconst

DESCRIPTION

       lumconstants  sets  the values of constants that are related to calcu-
       lating stellar luminosities for use in commands such as vista or view-
       star.

              msolunit
                 is the system mass unit in solar masses.

              kpcunit
                 is the system length unit in kiloparsecs.
