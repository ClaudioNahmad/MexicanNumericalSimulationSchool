NAME

       xrayload - read xray data

SYNOPSIS

       xrayload splinefile

DESCRIPTION

       xrayload  reads in the spline coeffients for interpolating X-ray lumi-
       nosity as a function of temperature.  The temperatures  are  given  in
       log(Kelvin), and the emissivity is in log(10^{-23} ergs/s/n_e/n_h/cc).

       The format of the file is:


              comment line

              redshift

              number_of_spline_points min_temperature delta_temperature

              number_of_bands

              y spline coefficients for band 1

              y'' spline coefficients for band 1

              y spline coefficients for band 2

              y'' spline coefficients for band 2
