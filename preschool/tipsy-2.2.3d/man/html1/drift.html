NAME

       hubble - extrapolate particle positions in a box

SYNOPSIS

       drift box-number particle delta-t

DESCRIPTION

       drift  drifts  the  particles  of  type particle in box box-number for
       delta-t using the particle's velocity.  This  command  is  useful  for
       interpolating  between time frames when making movies.  To move parti-
       cles forward in time use a positive value for delta-t and to move par-
       ticles  backward in time use a negative value for delta-t.  NOTE: this
       command permanently changes the velocities of all the particles in box
       box-number.  To undo a drift command repeat the command with a delta-t
       of equal magnitude but of opposite sign.  If the box box-number is not
       loaded an error is generated.

OPTIONS

   Particle
       The possible choices for particle are:

              all
                 Adjust  the  velocities of all the particles in box box-num-
                 ber.

              baryon
                 Adjust the velocities of all the gas and star  particles  in
                 box box-number.

              dark
                 Adjust  the  velocities of only the dark matter particles in
                 box box-number

              gas
                 Adjust the velocities of only the gas particles in box  box-
                 number.

              star
                 Adjust the velocities of only the star particles in box box-
                 number.
