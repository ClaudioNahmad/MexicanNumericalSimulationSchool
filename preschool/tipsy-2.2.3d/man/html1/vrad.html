NAME

       viewrad  - plot particles according to radial component of velocity or
       vector

SYNOPSIS

       viewall particle min max plot_type [ clip ]

ALIASES

       vrad

DESCRIPTION

       viewrad plots  particles  of  type  particle  (either  "gas",  "dark",
       "star",  or  "all")  in  the  active  box.   The particles are colored
       according to the radial component of either  the  particle's  velocity
       (if  plot_type  is  "velocity") or the vector loaded by readvector (if
       plot_type is "vector") using a rainbow colormap.  min and max set  the
       value of the minimum and maximum color.

SEE ALSO

       readvector(1) xrad(1), yrad(1), zrad(1)
