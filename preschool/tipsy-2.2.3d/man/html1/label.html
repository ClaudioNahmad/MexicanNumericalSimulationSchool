NAME

       label - place a label in the plot

SYNOPSIS

       label x-position y-position orientation text

DESCRIPTION

       label  places  the  label text at the position indicated by x-position
       and y-position.  The orientation of the label with respect to the des-
       ignated position is designated by an integer 1-9 as in the position of
       the numeric keys in a standard keypad.

       This is useful when making hardcopies using the hard command.

SEE ALSO

       colorbarlabel(1), hard(1)

BUGS

       Repeated use of the label command does not erase the  previous  label.
       To  erase  the  previous  label redraw the whole screen using the plot
       command.
