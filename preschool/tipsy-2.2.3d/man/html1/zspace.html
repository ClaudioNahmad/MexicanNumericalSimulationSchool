NAME

       zspace - move particles to redshift space in a box

SYNOPSIS

       zspace box-number hubble-constant x-center y-center z-center

DESCRIPTION

       zspace  adds  the velocity divided by the hubble constant to particles
       in box box-number centered on  the  coordinates  (x-center,  y-center,
       zcenter)  with  a Hubble constant of hubble-constant.  This command is
       useful for looking at a particle distribution as it would be  seen  in
       redshift  space.  NOTE: this command permanently changes the positions
       of all the particles in box If the box box-number  is  not  loaded  an
       error is generated.
