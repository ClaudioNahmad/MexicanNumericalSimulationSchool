NAME

       shell - execute a system command

SYNOPSIS

       shell <system command>

DESCRIPTION

       shell  allows the user to execute a system command from the TIPSY com-
       mand line.  This is useful when making movies from long macro  scripts
       using the macro command.

SEE ALSO

       macro(1)
