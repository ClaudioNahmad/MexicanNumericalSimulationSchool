NAME

       uvfield  -  set  the  values of constants related to the uv background
       field

SYNOPSIS

       uv uv_type jnu21 alphaj

ALIASES

       uv

DESCRIPTION

       jnu21 is the flux of the background radiation field in units of  10-21
       ergs-1 cm-2 Hz-1 sr-1.

       alphaj
          is    the    slope   of   the   UV   background   spectrum,   j   =
          jnu21*(nu/nu_L)**(-alphaj).
