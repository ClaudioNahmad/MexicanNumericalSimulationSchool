NAME

       openascii - open an ascii file

SYNOPSIS

       openascii filename

DESCRIPTION

       openascii  is a command that opens an ascii file of the name, filename
       for use by commands like readascii.  If an ascii file is already  open
       an error is generated.  Ascii files can be closed using the closeascii
       command.  Binary files are opened and closed using the openbinary  and
       closebinary commands.

SEE ALSO

       closeascii(1), closebinary(1), openbinary(1), readascii(1)
