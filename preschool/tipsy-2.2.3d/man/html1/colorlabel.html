NAME

       colorbarlabel - place labels over the color bar

SYNOPSIS

       colorbarlabel position text

ALIASES

       colorlabel, barlabel

DESCRIPTION

       colorbarlabel places the label text above the colorbar at the location
       position.  Will not function for all type plots.  This command is use-
       ful for making photographs or hardcopies.

OPTIONS

   Position
       The possible choices for position are:

              left
                 Write  text over the left edge of the colorbar.  The same as
                 using l.

              middle
                 Write text over the center of the  colorbar.   The  same  as
                 using m.

              right
                 Write text over the right edge of the colorbar.  The same as
                 using r.
