NAME

       resetcolor - reset the colormap

SYNOPSIS

       resetcolor [ where ]

ALIASES

       rcol

DESCRIPTION

       resetcolor resets the colormap to its original value.  If the optional
       argument where is given as a value between 0.0 and 1.0,  the  colorbar
       will be scrolled by the corresponding fraction.

       clearall(1),  clearrot(1),  resetcolor(1), unloadall(1), unmarkall(1),
       unzoom(1)
