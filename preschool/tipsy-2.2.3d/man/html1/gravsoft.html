NAME

       gravsoft - set the gravitational softening for the dark matter

SYNOPSIS

       gravsoft eps

ALIASES

       eps

DESCRIPTION

       gravsoft  sets the value of the gravitational softening length for the
       dark particles to eps for use in commands such as rotationcurve.   The
       softening length is in system units.
