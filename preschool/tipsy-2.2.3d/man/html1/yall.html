NAME

       yall - do an all plot projected along the y axis

SYNOPSIS

       yall

DESCRIPTION

       yall  is  a command that redraws the plot showing all particles in the
       simulation and projects it along the y axis.  This  command  takes  no
       arguments.   It also initializes box 0 when first loading a file using
       openbinary and loadbinary.

SEE ALSO

       xall(1), zall(1), loadbinary(1), openbinary(1)
