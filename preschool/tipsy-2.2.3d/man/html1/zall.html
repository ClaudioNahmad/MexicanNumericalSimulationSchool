NAME

       zall - do an all plot projected along the y axis

SYNOPSIS

       zall

DESCRIPTION

       zall  is  a command that redraws the plot showing all particles in the
       simulation and projects it along the z axis.  This  command  takes  no
       arguments.   It also initializes box 0 when first loading a file using
       openbinary and loadbinary.

SEE ALSO

       xall(1), yall(1), loadbinary(1), openbinary(1)
